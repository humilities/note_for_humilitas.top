# Parkour Design
我们考察三个位移方向的位移向量，可以知道v1+v2=2v3. 在线性代数上三者线性相关。
在此基础上我们可以得出两个条件。
首先根据线性规划，我们可以得出目标位置与斜率之间的关系
$-\frac{x}{4} \le y \le \frac{x}{2}$
对上述的结论，在线性相关的基础上，通过基变换，我们可以找到
$x+y \equiv 0 \pmod 3$
