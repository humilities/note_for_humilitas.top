by working backwords.

With the reverse, we iterate through its element. If an element is already contained in Q,
ignore it and otherwise we append it to the end of Q. So we need a array to store the
elements that have visited.

Thus, at each step we choose the array with the smallest lexicographically among the remain
ones.
